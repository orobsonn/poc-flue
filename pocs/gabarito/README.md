# gabarito

Avaliador autônomo de agentes em domínio de julgamento. POC do monorepo `poc-flue`.

**Tese**: aplica literal a metodologia de Decision Log + Classificação de Ground Truth do vault de Rob Lins, materializada em Flue como sistema agêntico que avalia outros sistemas agênticos e propõe ajustes via PR no GitHub.
